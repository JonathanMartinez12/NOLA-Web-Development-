# Portfolio Images

This directory contains images for portfolio projects displayed on the portfolio page.

## Required Images

Please add the following images to this directory:

1. **krewe-of-christmas.png** - Screenshot or image of the Krewe of Christmas website
2. **mendez-income-tax.png** - Screenshot or image of the Mendez Income Tax website
3. **fanns-cleaning-services.png** - Screenshot or image of Fann's Cleaning Services website

## Image Guidelines

- **Format**: PNG or JPG
- **Recommended dimensions**: 1200x800px (3:2 aspect ratio)
- **File size**: Keep under 500KB for optimal performance
- **Content**: Full-page screenshot or hero section of the website
- **Quality**: High-resolution, clear, and professional-looking

The images will be displayed in the portfolio cards with a gradient overlay on hover.
